# Mewsik Studio

Музыкальный редактор в кошачьем стиле. Цель проекта - создавать музыку из кошачьих звуков: из этих коротких звуков собираются паттерны, а из паттернов - целые треки.

## Описание MVP

Mewsik Studio состоит из двух связанных экранов: редактор паттернов и главный редактор. В редакторе паттернов из кошачьих звуков собираются короткие фрагменты, а в главном редакторе из этих паттернов выстраивается полноценная композиция.

### Редактор паттернов

- Библиотека котов, разбитых по ролям: Lead, Bass, Pad и Drums. Выбранный кот отображается в боковом меню со своим набором звуков.
- Звуки перетаскиваются из меню на дорожки секвенсора. Уже размещённые блоки тоже можно перетаскивать на другие клетки, а кликом - удалять.
- Избранные звуки сохраняются в список избранного для быстрого доступа.
- Кнопка Play запускает аоспроизведение паттерна по кругу до нажатия кнопки Stop.
- Темп задаётся счётчиком BPM и влияет на скорость проигрывания.
- Pad-звуки растягиваются на весь такт независимо от темпа, они всегда завершаются только к концу паттерна и тут же стартуют заново.
- У каждой дорожки есть своя громкость - регулируется ползунком.

### Главный редактор

- Сборка музыки из сохранённых паттернов: готовые фрагменты размещаются на дорожках в главном редакторе и складываются в трек.
- Есть переключение между проектами, можно вести несколько проектов и переходить между ними.
- Сохранение проектов и паттернов между сессиями - данные не теряются после закрытия приложения.

## Технологии

- React + Vite - UI и вёрстка
- Zustand - управление состояниями
- Tone.js - аудио-движок и любые манипуляции со звуком
- dnd-kit - перетаскивание блоков мышкой
- IndexedDB + Dexie.js - хранение проектов, паттернов и избранного между сессиями
